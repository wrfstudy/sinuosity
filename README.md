# tc-sinuosity

Code for the analysis in the paper:

**Climate-Driven Straightening of Western North Pacific Tropical Cyclone Trajectories**

## Description

This repository contains the Python scripts used to analyze the changes in tropical cyclone trajectory sinuosity over the Western North Pacific.

The code calculates tropical cyclone trajectory sinuosity and analyzes its relationship with environmental conditions.

## Data

The analysis uses publicly available datasets:

- IBTrACS tropical cyclone best-track data  
- ERA5 reanalysis data  
- ADT-HURSAT tropical cyclone dataset  
- CMIP6 HighResMIP tropical cyclone tracks

Please refer to the Data Availability Statement in the manuscript for detailed access links.

## Requirements

The code is written in Python. Common scientific libraries are required, including:

- numpy  
- pandas  
- xarray  
- matplotlib  
- scipy

## Usage

Run the main analysis scripts to reproduce the results presented in the manuscript.
